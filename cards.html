Cards

Color brings a design to life. It is used to express emotion and tone, as well as place emphasis and create associations. Color should only be used to attract attention.

Card that doesn't feature any images.

This card design allows for the inclusion of images, although it doesn't conform to the predefined image layouts such as top, left, or right alignment. 

                                        <-- No Image Card -->
                                        <div class="card card-shadow-dark">
                                            <div class="card-body">
                                                <h4 class="card-title"></h4>
                                                <div class="line"></div>
                                            </div>
                                        </div>

Notes

Encloses the given content within a specially styled container known as a Card.

If an imageUrl is provided, the Card can style an image in one of three positions: top, left, or right.

If additional padding is required, it should be explicitly defined using padding utility classes: containerClasses for the Card, contentClasses for any child elements, and imageClasses.
